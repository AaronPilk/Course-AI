# Clearances + communication

## Phraseology

Standard format from AIM (Aeronautical Information Manual):
- Aircraft type + call sign.
- Current state.
- Request.

Example: "Bay Approach, Cessna 12345, 5,000, request flight following to Sacramento."

For: clear, efficient comm.

## Aircraft call signs

- N-numbers (US): "November 12345" or "Skyhawk 12345."
- Airline: "United 245."
- Military: "Reach 1234" (transport), "Hawk 21" (fighter).

Phonetic alphabet for letters.

For: identification.

## Standard clearance format

ATC clearance components (CRAFT):
- **C**learance limit (destination).
- **R**oute.
- **A**ltitude.
- **F**requency (departure).
- **T**ransponder code.

Example: "Skyhawk 12345 cleared to Sacramento via direct SAC, climb 5,000, expect 8,000 ten minutes after departure, contact departure 124.4, squawk 4321."

For: IFR clearance structure.

## Read-back requirements

Must read back:
- Runway assignments.
- Altitude clearances.
- Heading vectors.
- Approach clearances.
- Hold short instructions.

Failure to read back = controller assumes you didn't hear.

For: safety.

## Hot mic etiquette

Wait for clear frequency. Listen before transmitting. Brief + clear.

Avoid:
- Excessive chatter.
- Stepping on others.
- Open mic stuck.

For: shared frequency.

## VFR comm minimums

VFR pilot communications:
- Class B: full ATC clearance.
- Class C: establish two-way comm.
- Class D: two-way comm (no clearance needed).
- Class E/G: optional (CTAF at uncontrolled).

For: required minimum interaction.

## Emergency comms

- 121.5 MHz "Guard" frequency: universal emergency.
- ATC always monitoring.
- Squawk 7700 (emergency), 7600 (lost comm), 7500 (hijack).

For: when things go wrong.

## Lost comm procedures

If radio fails on IFR:
- Squawk 7600.
- Continue to destination per clearance.
- Last-assigned route + altitude.
- Expect landing per cleared approach.

If VFR: navigate around controlled airspace or land at nearest airport.

For: emergency rules.

## Read-back, hear-back

Best practice both ways:
- Pilot reads back ATC instruction.
- ATC verifies if incorrect.
- If silent → both move on.

Subtle errors detected.

For: error catching.

## VFR communication

Less formal than IFR. Position reports:
- "Skyhawk 12345, 10 miles south of Stockton, 4,500, inbound for landing."

Closing the loop on flight following: "Sky 12345, cancel flight following."

For: VFR style.

## Approach communication

```
ATC: "Skyhawk 12345, descend and maintain 3,000, turn left heading 270, cleared ILS runway 28R approach."
Pilot: "Down 3,000, left heading 270, cleared ILS 28R, Skyhawk 12345."
```

Precise read-back.

For: instrument approach.

## CRM in cockpit

Crew Resource Management:
- Pilot flying + pilot monitoring.
- Cross-check + verify.
- Speak up if uncomfortable.
- Standard callouts ("1,000 to go," "500," "100 above").

For: airline safety culture.

## Mistakes to avoid

- **Mumbled call signs.** ATC misroutes.
- **Wrong frequency.** Loss of contact.
- **Forgetting to acknowledge.** Controller repeats.

## Summary

- Phraseology standardized in AIM.
- IFR clearance: CRAFT format.
- Read-back required for critical items.
- 121.5 + 7700 for emergencies.
- 7600 lost comm; 7500 hijack.
- CRM in multi-pilot cockpits.
