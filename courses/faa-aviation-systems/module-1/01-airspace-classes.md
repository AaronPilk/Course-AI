# Class A through G airspace

## Why classes exist

Different airspace has different rules for safety + ATC workload. Higher complexity = stricter rules.

For: framework.

## Class A

- 18,000 ft MSL to FL600 (60,000 ft).
- IFR only; no VFR allowed.
- ATC clearance required.
- Most airliners cruise here.

For: high-altitude jet traffic.

## Class B

- Surrounds busiest airports (JFK, LAX, ATL, ORD).
- Surface to 10,000 ft typically.
- Shape: "upside-down wedding cake."
- VFR allowed with ATC clearance.
- Mode C transponder required.

Specific clearance: "Cleared to enter Class Bravo."

For: busy hub airports.

## Class C

- Around medium-busy airports (radar-equipped).
- Surface to 4,000 ft typical.
- Two-way radio + transponder required.
- No specific clearance needed (just establish two-way comms).

For: regional airports.

## Class D

- Around airports with operating control tower.
- Surface to 2,500 ft typical.
- Two-way radio required.
- When tower closes: reverts to Class E or G.

For: small/medium towered fields.

## Class E

- Most controlled airspace.
- Above 1,200 ft AGL nationwide (most areas).
- VFR weather minimums required.
- No two-way radio required.

For: en-route VFR + IFR.

## Class G

- Uncontrolled airspace.
- Surface to base of Class E (usually 1,200 ft AGL).
- Most permissive (still need basic VFR weather).
- Rural areas, low altitudes.

For: where ATC doesn't actively manage.

## VFR weather minimums

Class A: N/A (IFR only).
Class B: 3 statute miles vis + clear of clouds.
Class C/D/E (below 10k): 3 SM vis + 500 below/1,000 above/2,000 horizontal from clouds.
Class G: more permissive, varies with altitude.

For: legal flight planning.

## Equipment requirements

- **Transponder.** All Class A/B/C; certain Class E.
- **ADS-B Out.** Required in most controlled airspace since 2020.
- **Mode C / S.** Transponder modes for altitude reporting.

For: hardware compliance.

## Chart symbology

Airspace shown on aeronautical charts:
- Class B: solid blue line.
- Class C: solid magenta line.
- Class D: dashed blue line.
- Class E: shaded blue or magenta.
- Class A: not depicted (always 18,000+).

For: chart reading.

## Mistakes to avoid

- **Entering Class B without clearance.** FAA violation.
- **Wrong transponder code.** Confuses ATC.
- **Misreading chart boundaries.** Inadvertent airspace bust.

## Summary

- Classes A-G define rules + ATC involvement.
- A: jet altitude IFR-only.
- B/C/D: airports of different busyness.
- E: most controlled, no comm required.
- G: uncontrolled.
- VFR weather mins vary by class.
