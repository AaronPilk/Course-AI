# VFR vs IFR

## VFR (Visual Flight Rules)

Pilot navigates by looking outside:
- See + avoid other aircraft.
- See terrain.
- Read landmarks.

Requires weather minimums (visibility + cloud clearance).

For: clear weather flying.

## IFR (Instrument Flight Rules)

Pilot navigates by instruments only:
- Through clouds.
- At night over water.
- Anywhere weather marginal.

ATC clearance required; constant communication.

For: bad weather + airline flying.

## VFR weather minimums

Class B: 3 SM visibility + clear of clouds.
Class C/D/E (<10k ft): 3 SM + 500/1000/2000 cloud clearance.
Class G (day): 1 SM + clear of clouds (low altitude).
Class G (night): 3 SM + 500/1000/2000.

For: legal VFR.

## IFR requirements

Pilot:
- Instrument rating.
- Current (6 approaches in last 6 months).
- Filed IFR flight plan.

Aircraft:
- Properly equipped (IFR-certified avionics).
- Current pitot-static check (24 months).
- Current transponder check (24 months).

For: legal IFR.

## Filing IFR

```
Aircraft: Cessna 172, N12345
Type: I (IFR)
Equipment: G (GPS) or H (RNAV)
Speed: 110 kts
Departure: KSFO 1500Z
Route: SFO V25 SAC V107 RNO
Altitude: 8000 ft
Destination: KRNO 1700Z
Time enroute: 02:00
Fuel: 04:30
Alternate: KMMH
PIC: John Doe / phone
```

Filed via 1800wxbrief, ForeFlight, etc.

For: IFR flight plan.

## ATC clearance

For IFR departure:
```
"Skyhawk 12345, cleared to Reno via SFO Departure 5, V25 SAC V107 RNO,
maintain 5000, expect 8000 ten minutes after departure, departure frequency 124.4,
squawk 4321."
```

Pilot reads back; must comply.

For: IFR coordination.

## VFR flight following

Optional ATC service for VFR:
- Traffic advisories.
- Workload-permitting.
- Not separation service.

Request from approach control.

For: VFR situational awareness.

## VFR over the top

Take off VFR, climb above cloud layer, fly with clear conditions, return. Legal but trapped if destination is socked in.

Risky for non-IFR pilots.

For: edge case.

## Pop-up IFR

VFR pilot encountering deteriorating weather can request IFR clearance from ATC:
- "Request IFR clearance to KSFO."
- Avoids visual flight into IMC (a leading cause of accidents).

For: emergency option.

## Marginal VFR (MVFR)

Borderline conditions:
- Visibility 3-5 SM.
- Ceiling 1,000-3,000 ft.

Legal but caution; can deteriorate quickly.

For: judgment call.

## VFR night

Allowed in U.S. but requires:
- Pilot night currency (3 takeoffs + landings in 90 days for passengers).
- Aircraft lighting.
- Higher weather mins in Class G.

For: special category.

## When IFR best

- Cloud layers.
- Night.
- Over water.
- High-traffic areas.
- Convective weather (with ATC routing around).

For: practical IFR.

## When VFR best

- Clear weather.
- Scenic flying.
- Low altitude / pattern work.
- Short hops.
- Cost (no IFR currency cost).

For: practical VFR.

## Mistakes to avoid

- **VFR into IMC.** Deadly statistically.
- **Skipping go/no-go.** Get-there-itis.
- **Stale weather brief.** Plan with fresh data.

## Summary

- VFR = visual; IFR = instruments.
- VFR has weather minimums.
- IFR requires rating + currency + ATC.
- Flight following for VFR awareness.
- Marginal VFR = warning sign.
