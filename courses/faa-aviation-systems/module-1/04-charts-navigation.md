# Charts + navigation

## Chart types

- **Sectional.** 1:500,000 scale; VFR navigation. Most used.
- **TAC (Terminal Area Chart).** 1:250,000; busy urban airspace detail.
- **WAC.** Discontinued; was 1:1,000,000.
- **IFR Low / High Enroute.** For IFR cruise.
- **TPP (Terminal Procedures).** Approach + departure procedures.
- **Airport Diagram.** Detailed airport layout.

For: chart selection.

## Sectional anatomy

- **Color shading.** Terrain elevation.
- **Magenta dashed lines.** Class E surface boundaries.
- **Blue solid lines.** Class B boundaries.
- **Magenta solid lines.** Class C.
- **Blue dashed.** Class D.
- **Obstacles.** Towers, antennas with heights.
- **Airport symbols.** Different for paved, grass, private.

For: read at a glance.

## VFR cruising altitudes

Magnetic course determines altitude:
- 0-179°: odd thousands + 500 (3,500; 5,500).
- 180-359°: even thousands + 500 (4,500; 6,500).

Above 3,000 ft AGL only.

For: vertical separation by direction.

## VORs

VHF Omnidirectional Range:
- Ground stations broadcast radials.
- Receiver shows azimuth from station.
- Multiple VORs = triangulation.
- Being phased out as GPS replaces.

For: legacy navigation.

## GPS

- Most modern primary nav.
- WAAS GPS for IFR approaches.
- ADS-B Out uses GPS for position.
- More precise than VOR/DME.

For: current navigation.

## ADS-B (Automatic Dependent Surveillance-Broadcast)

- **ADS-B Out.** Transmits position to ATC + other aircraft.
- **ADS-B In.** Receives weather + traffic data.
- Required since 2020 in most controlled airspace.
- ~100x more accurate than radar.

For: modern surveillance.

## E6B flight computer

Manual whiz-wheel for:
- Time/distance/speed.
- Wind correction angle.
- Fuel consumption.
- Density altitude.

Replaced by apps but required for written exam.

For: legacy + theory.

## Foreflight + Apps

iPad-based EFB (Electronic Flight Bag):
- Charts.
- Weather.
- Flight planning.
- ADS-B traffic + weather.
- Synthetic vision.

Game-changer for GA.

For: modern cockpit.

## Pilotage + dead reckoning

- **Pilotage.** Visual landmarks.
- **Dead reckoning.** Heading + speed + time = position.

Required teaching even with GPS. Backup if GPS fails.

For: fundamentals.

## RNAV / RNP

Area Navigation:
- Fly point-to-point (not VOR-to-VOR).
- RNAV 1, RNAV 2, RNP authorizations.
- More efficient routes.

NextGen uses RNAV extensively.

For: modern routing.

## Navigation log

Pre-flight planning document:
- Route segments.
- Headings + altitudes.
- Distances + times.
- Fuel calculations.
- Checkpoints.

Required for cross-country.

For: organized planning.

## Mistakes to avoid

- **Trust GPS blindly.** Cross-check with chart.
- **Stale charts.** Updated every 56 days.
- **Misread magnetic course.** Wrong altitude.

## Summary

- Sectionals for VFR; IFR charts for IFR.
- VFR cruising altitudes based on magnetic course.
- VOR legacy; GPS + WAAS + ADS-B modern.
- Pilotage + dead reckoning as backup.
- ForeFlight + EFBs revolutionized cockpit.
