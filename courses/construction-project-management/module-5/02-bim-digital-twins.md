# BIM + digital twins

## What BIM is

Building Information Modeling: 3D model + metadata for entire building:
- Geometry.
- Properties of each component.
- Quantities.
- Schedules.
- Costs.

Single source of truth.

For: integrated design.

## BIM levels

- **Level 0.** Paper drawings.
- **Level 1.** 2D CAD.
- **Level 2.** 3D BIM in silos (architect/MEP/structural separate).
- **Level 3.** Integrated BIM, shared model.
- **Level 4.** + lifecycle data.
- **Level 5.** + cost + carbon.

Most projects 2-3 today.

For: maturity.

## Software

- **Revit (Autodesk).** Most popular architecture/MEP.
- **ArchiCAD.** Architecture-focused.
- **Tekla Structures.** Structural steel detailing.
- **Navisworks.** Clash detection + 4D.
- **BIM 360 / Construction Cloud.** Collaboration platform.

For: tool selection.

## Clash detection

Run automated checks:
- HVAC duct hitting beam.
- Plumbing pipe through column.
- Door clash with wall.

Catch in model before built → save $$$.

For: pre-construction value.

## 4D BIM

Add time:
- Schedule activities linked to model elements.
- Visualize construction sequence.
- Identify schedule risks.

Synchro, Navisworks Timeliner.

For: schedule visualization.

## 5D BIM

Add cost:
- Quantities auto-update.
- Cost tracking real-time.
- Value engineering options.

Used in IPD + sophisticated projects.

For: cost integration.

## 6D + 7D

- **6D.** Sustainability / energy.
- **7D.** Operations + maintenance.

Less standardized; some adopters.

For: extended dimensions.

## Digital twin

BIM model that lives:
- Sensors update model in real-time.
- IoT integration.
- Operations data linked.

Used post-construction.

For: facility management.

## Examples

- **Heathrow Airport.** Digital twin for facility management.
- **Hudson Yards.** Massive BIM coordination.
- **Crossrail (UK).** Pioneer BIM Level 2 mandate.

For: real applications.

## BIM benefits

Studies show:
- 7% time savings.
- 20% cost savings (large projects).
- 50% fewer change orders.
- Better safety planning.

But: requires investment + training.

For: ROI.

## Challenges

- Steep learning curve.
- Subcontractors not all on board.
- Software licensing costs.
- Hardware requirements.
- Standards evolving.

For: realistic adoption.

## VR + AR

- **VR.** Walkthrough before construction.
- **AR.** Overlay model on actual site (HoloLens, iPad).

Increasing adoption for clients + field.

For: visualization tech.

## COBie

Construction Operations Building Information Exchange:
- Standard format for handover data.
- Owner gets digital asset.
- Used for facility management.

Government often requires.

For: handover standard.

## IFC

Industry Foundation Classes:
- Open standard BIM file format.
- Interoperability between tools.
- Vendor-neutral.

ISO 16739.

For: openness.

## BIM mandates

Required by:
- UK Government Construction Strategy (2016+).
- Many European countries.
- US: GSA, Veterans Affairs, NYC DOB.

Growing globally.

For: regulatory trend.

## Mistakes to avoid

- **Buying software without process.** Underused.
- **Single discipline only.** Misses coordination value.
- **No handover plan.** Data lost.

## Summary

- BIM = 3D model + metadata; Levels 0-5.
- Revit + Navisworks dominant.
- Clash detection prevents field issues.
- 4D = schedule, 5D = cost.
- Digital twins = live model for operations.
- IFC + COBie open standards.
