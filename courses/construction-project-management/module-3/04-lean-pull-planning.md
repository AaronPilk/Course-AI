# Lean construction + Pull Planning

## What lean is

Toyota Production System adapted to construction:
- Reduce waste.
- Maximize value.
- Continuous improvement (Kaizen).
- Respect for people.

For: efficiency mindset.

## Wastes (8 types)

- **Defects.** Rework.
- **Overproduction.** Built too early.
- **Waiting.** Delays.
- **Non-utilized talent.**
- **Transportation.** Moving materials.
- **Inventory.** Excess stockpiled.
- **Motion.** Unnecessary worker movement.
- **Excess processing.** Doing more than required.

Acronym: DOWNTIME.

For: identifying waste.

## Last Planner System

Pull-based planning by Lean Construction Institute (LCI):

1. **Master schedule.** High-level.
2. **Phase schedule.** Pull plan by phase.
3. **Lookahead.** 6-week constraint-free planning.
4. **Weekly Work Plan.** Make-ready + commit.
5. **Daily check-in.** Stand-ups.

For: lean rhythm.

## Pull planning

Vs. traditional push:
- Start at end date.
- Work backward.
- Each trade pulls predecessors.
- Identifies handoffs explicitly.

Done collaboratively in pull-planning sessions.

For: better coordination.

## PPC (Percent Plan Complete)

Each week:
- Plan tasks for next week.
- Track which actually complete.
- PPC = completed / planned.

Target: 80%+ PPC.

Improves trade reliability.

For: metric.

## Constraints

Pre-week:
- Identify what could block tasks.
- Resolve before commitment.

Types: design, material, equipment, prerequisites, permits.

If can't resolve, don't commit.

For: avoiding rework.

## A3 problem solving

Toyota A3 = 11x17 paper for structured problem solving:
- Background.
- Current state.
- Goal.
- Analysis.
- Countermeasures.
- Plan.

One page = clear thinking.

For: solving recurring issues.

## Big rooms

Co-locate team:
- Designers + builders + subs in one space.
- Live model on screens.
- Whiteboard pull plans.

Reduces RFIs + accelerates decisions.

For: collaboration.

## Prefab + modular

Build off-site:
- Wall panels.
- MEP racks.
- Bathroom pods.
- Hotel rooms.

Reduces on-site time + waste; increases quality.

Growing 20%+ annually.

For: off-site construction.

## IPD + lean

IPD contracts naturally support lean:
- Shared risk/reward.
- Open book.
- Collaborative incentive.

For: contract enabling lean.

## Resistance to lean

Common barriers:
- "We've always done it this way."
- Adversarial culture.
- Lump-sum contracts discourage transparency.
- No top-management buy-in.

Lean requires culture change.

For: realistic.

## Tools

- **VPL (Visual Production Levels).** Daily commitments.
- **Kanban.** Visual workflow.
- **5S.** Sort/Set/Shine/Standardize/Sustain.
- **Pareto charts.** Top causes of issues.

For: practical implementation.

## Results

Lean teams typically achieve:
- 10-30% schedule reduction.
- 5-20% cost savings.
- 80%+ PPC.
- Higher worker satisfaction.

Best results in repeat-client / IPD setups.

For: ROI.

## Mistakes to avoid

- **Tools without culture.** Tools alone won't fix.
- **No leadership commitment.** Effort fizzles.
- **Implementing all at once.** Pilot one project first.

## Summary

- Lean = reduce waste, maximize value.
- 8 wastes: DOWNTIME acronym.
- Last Planner System + Pull Planning.
- PPC tracks reliability; 80%+ target.
- Prefab + big rooms = practical applications.
- Culture change > tool deployment.
