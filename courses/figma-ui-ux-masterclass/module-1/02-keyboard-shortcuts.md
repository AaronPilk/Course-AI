# Keyboard shortcuts that matter

## Why memorize them

Mouse work is slow. Designers fluent in shortcuts produce 2-3× faster than those who menu-hunt. The investment is small (a week of conscious use) and the payback is permanent.

You don't need to memorize hundreds. The 30-40 below cover ~90% of routine work.

## The essentials — file & navigation

| Action | Mac | Windows |
|--------|-----|---------|
| New file | Cmd+N | Ctrl+N |
| Save (auto-saves) | Cmd+S | Ctrl+S |
| Undo | Cmd+Z | Ctrl+Z |
| Redo | Cmd+Shift+Z | Ctrl+Y |
| Zoom to fit | Shift+1 | Shift+1 |
| Zoom to 100% | Shift+0 | Shift+0 |
| Zoom to selection | Shift+2 | Shift+2 |
| Zoom in / out | Cmd +/- | Ctrl +/- |
| Pan | Spacebar+drag | Spacebar+drag |
| Show / hide left panel | Cmd+\ | Ctrl+\ |
| Show / hide UI | Cmd+. | Ctrl+. |

Spacebar+drag for panning is universal. Use it instead of dragging the scrollbar.

## Creating elements

| Action | Shortcut |
|--------|----------|
| Frame | F |
| Rectangle | R |
| Ellipse | O |
| Line | L |
| Arrow | Shift+L |
| Text | T |
| Pen | P |
| Pencil | Shift+P |
| Comment | C |

Press the key, then click + drag to draw. With nothing selected, the new element appears on the canvas at default size.

## Selection

| Action | Shortcut |
|--------|----------|
| Select all | Cmd+A |
| Deselect | Esc |
| Enter (drill into nested) | Enter |
| Exit (drill out) | Esc |
| Select layer (any depth) | Cmd/Ctrl+click |
| Add to selection | Shift+click |
| Select same fill | Right-click → Select all with same fill |

Cmd/Ctrl+click is your most important selection shortcut. Without it, you click-click-click to get to a nested layer.

## Editing

| Action | Shortcut |
|--------|----------|
| Copy | Cmd+C |
| Paste | Cmd+V |
| Paste in place | Cmd+Shift+V |
| Duplicate | Cmd+D |
| Delete | Delete |
| Rename layer | Cmd+R |

Cmd+D is huge — duplicates the selection with the same offset as the previous duplicate. Drag once, then Cmd+D repeatedly = even rows of duplicates.

## Group & frame

| Action | Shortcut |
|--------|----------|
| Group | Cmd+G |
| Ungroup | Cmd+Shift+G |
| Frame selection | Cmd+Alt+G |
| Auto Layout | Shift+A |
| Remove Auto Layout | Shift+Alt+A |

Shift+A is the workhorse. Select multiple layers; Shift+A to wrap in Auto Layout frame.

## Layout & alignment

| Action | Shortcut |
|--------|----------|
| Align left | Cmd+Alt+A → Left |
| Center horizontally | Cmd+Alt+H |
| Center vertically | Cmd+Alt+V |
| Distribute evenly | Cmd+Alt+T (top), Cmd+Alt+B (bottom) |

The Align panel in the right panel + clicks is also fine; learn one or the other.

## Text

| Action | Shortcut |
|--------|----------|
| Bold | Cmd+B |
| Italic | Cmd+I |
| Underline | Cmd+U |
| Increase size | Cmd+Shift+> |
| Decrease size | Cmd+Shift+< |

For frequent text work, learn these.

## Style copy / paste

| Action | Shortcut |
|--------|----------|
| Copy properties | Cmd+Alt+C |
| Paste properties | Cmd+Alt+V |

Select the styled element; Cmd+Alt+C; select the destination; Cmd+Alt+V. Properties (fill, stroke, effects) transfer.

## Components

| Action | Shortcut |
|--------|----------|
| Create component | Cmd+Alt+K |
| Detach instance | Cmd+Alt+B |
| Swap instance | Cmd+Alt+B (then select replacement) |

Components are central to design systems. Cmd+Alt+K to create from selection.

## Show measurements

Hold Alt/Option while hovering. Spacing measurements appear between hovered element and selected element. Indispensable for ensuring consistent spacing.

## Distance to neighbors

Select an element; hold Alt; hover over another. Distance shown. Faster than computing offsets manually.

## Find shortcut for any action

Cmd+/ opens the quick action menu. Type any action name; see its shortcut. Use this to discover shortcuts you don't know.

## Building shortcut habits

- **Don't try to memorize all at once.** Pick 5; use them all week; add 5 more.
- **Print a cheat sheet.** Put it next to your monitor for first month.
- **Watch your own work.** Notice menu-hunting; learn the shortcut instead.
- **Quick action menu (Cmd+/) is your friend.** Discovers shortcuts as you work.

After a month: you'll do mouse-vs-keyboard automatically. Productivity jumps measurably.

## Mistakes to avoid

- **Menu-hunting forever.** Spend the week to learn shortcuts.
- **Memorizing shortcuts you don't need.** Focus on what you do daily.
- **Trying to learn all at once.** Incremental; sustainable.
- **Different shortcuts on Mac vs Windows.** Use Figma on your primary OS; muscle memory builds.

## Summary

- Mouse work is slow; shortcuts compound speed daily.
- Memorize ~30 essentials covering navigation, creation, editing, selection, layout.
- Spacebar+drag, Cmd/Ctrl+click, Cmd+D, Shift+A, Alt+hover are top 5.
- Cmd+/ discovers shortcuts as you work.
- Incremental learning; expect 2-3× speed gain after a month.
