# Style libraries and reference workflows

## Building your prompt library

Save what works:
- **Successful prompts.** Copy from generation; annotate why it worked.
- **Failed prompts + diagnosis.** Why didn't this work? Don't repeat.
- **Style descriptors.** "Cinematic," "moody," "isometric" — collect.
- **Camera + lighting stacks.** "Shot on Sony A7R IV, 50mm f/1.4, golden hour."

Tools: Notion, simple text files, prompt-management tools (PromptHero, PromptBase).

## Style categorization

Organize by:
- **Photographic.** Realistic / film / portrait / street / product.
- **Illustration.** Cartoon / line / digital painting / watercolor.
- **3D.** Octane render / Unreal Engine render / Blender.
- **Specific aesthetics.** Studio Ghibli / Pixar / 90s anime / vaporwave / cyberpunk.
- **Fine art.** Impressionism / Renaissance / Art Nouveau.

For: pick + apply per project.

## Style transfer prompts

Common patterns:
- "In the style of [style descriptor], [subject + scene]."
- "[Subject], [style descriptor] art."
- "[Subject] painted by [artist-equivalent style descriptor]."

Examples:
- "Watercolor painting of a fox in autumn forest."
- "Anime art style, young heroine standing on rooftop."
- "Concept art, sci-fi spacecraft, detailed digital painting."

For: invoke specific aesthetics.

## Reference images

Beyond prompts:
- **img2img.** Use existing image as starting point; prompt to transform.
- **IP-Adapter.** Use reference image as style guide.
- **ControlNet (next module).** Use reference for pose / depth / composition.

Reference images often more powerful than pure text for getting specific style.

## Civitai prompts library

Civitai.com is the central hub:
- Search models / LoRAs / embeddings.
- See generated images with their prompts.
- Copy prompts from images you like.
- Adapt to your needs.

For: learning + inspiration; remix others' prompts.

## Trending styles 2026

- **Cinematic photography.** Moody, color-graded, film-look.
- **Hyperreal portraits.** Sharp skin detail; backlit.
- **Anime mainstream.** Studio Ghibli-inspired; Makoto Shinkai aesthetic.
- **Cyberpunk / synthwave.** Neon, futuristic.
- **Isometric digital art.** Tilted 3D look, for games / illustration.
- **Cottagecore / pastoral.** Soft, painterly, rural.

For: contemporary work; match audience expectations.

## Avoiding artist names

Many image gen communities now discourage / filter artist names:
- Legal / ethical concerns about training data.
- Some artists have demanded removal.

Use general descriptors instead:
- Don't: "by Greg Rutkowski"
- Do: "fantasy concept art, dramatic lighting, oil painting style"

For: ethical practice + future-proofing prompts.

## Compound styles

Mixing styles:
- "Studio Ghibli style + cyberpunk aesthetic."
- "Watercolor + sci-fi."
- "Pixar 3D + medieval fantasy."

For: unique aesthetics; experimentation.

Some combinations work great; others mush. Iterate.

## Era / decade references

For temporal style:
- "1980s aesthetic, VHS quality."
- "1950s film noir."
- "1920s art deco."
- "Y2K aesthetic, early 2000s."

For: nostalgic / period-specific work.

## Genre references

For specific moods:
- "Horror movie still, dark atmosphere, dread."
- "Action movie poster, dramatic, vibrant."
- "Coming-of-age film, golden hour, warmth."

For: invoking film genre conventions.

## Lighting library

Build your lighting vocabulary:
- "Soft window light."
- "Hard top-down sun light."
- "Dramatic chiaroscuro."
- "Cinematic three-point lighting."
- "Volumetric god rays."
- "Neon underglow."
- "Bioluminescent ambient."
- "Candlelight, warm flickering."

Each invokes specific mood.

## Iteration workflow

For finding a good prompt:
1. Start with descriptive prompt.
2. Generate batch of 4.
3. Pick the closest to intent.
4. Note what's missing / wrong.
5. Adjust prompt.
6. Repeat.

Usually 3-10 iterations to nail a hero image.

## Sharing prompts

Communities:

For: feedback + inspiration; build network.

## Mistakes to avoid

- **No prompt library.** Forget what worked; repeat learning.
- **One style for all subjects.** Photoreal car + photoreal product + photoreal portrait = monotony.
- **Artist names lazy.** Use specific style descriptors.
- **Same prompt for different models.** Adapt.

## Summary

- Build personal prompt library; categorize by style; save what works.
- Civitai for model + prompt research.
- Use reference images (img2img, IP-Adapter, ControlNet) for specific styles.
- General style descriptors over artist names.
- Compound styles for unique aesthetics.
- Iterate; 3-10 rounds per hero image.
